# GoodFood System

Bem-vindo ao **GoodFood System**, uma aplicação moderna estruturada com backend em Laravel, frontend em Next.js e banco de dados PostgreSQL. O ambiente de desenvolvimento é totalmente conteinerizado utilizando Docker Compose, garantindo um setup rápido, previsível e isolado.

Este é o documento de entrada do projeto. Para aprofundar-se em como o sistema funciona ou em como contribuir, consulte a nossa documentação detalhada na pasta `docs/`.
